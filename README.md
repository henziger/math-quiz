# Math Quiz
A mathy slide quiz built with React.

[View Demo](https://henziger.se/math-quiz/)

Based on David Ra's [react-quiz](https://github.com/davidrayoussef/react-quiz). 

Getting Started
---------------

```shell
$ git clone https://github.com/henziger/math-quiz.git
$ cd math-quiz
$ npm install
$ npm start
```

App Structure
-------------

```
QuizApp
|── Settings
|
└── Quiz
    ├── QuestionList
    |   └── Question
    |       └── Answer
    └── Results
        └── Chart
```
